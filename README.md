# Glass Tavern

A character card manager for SillyTavern with a glassmorphism UI.  
Import, organize, search, tag, and export your character cards — no API key required.

## 🚀 Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages → Source** → select **GitHub Actions**
3. Push to `main` branch — auto-builds and deploys. No secrets needed.

## Development

```bash
npm install
npm run dev
```
